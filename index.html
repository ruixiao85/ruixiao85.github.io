ParenchymalAirspaceProfiling Software

For those new to the software, please click "Settings" from the menu and load "~test_small" or "~test_large" to see some examples.
Please feel free to report any bugs to Rui.

This is a Windows Presentation Foundation application written in C#. It requires .NET 4.6 framework to run properly, which is generally pre-installed in Windows 8 and 10.
